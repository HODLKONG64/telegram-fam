## SPECIAL CONDITIONS

### HIGH SIGNAL EVENT
Triggered when:
- multiple updates in one cycle
- large mention_count entities
- major drops or announcements

Effect:
- faster pacing
- more urgency
- multiple threads colliding

---

### LOW SIGNAL STATE
Triggered when:
- little or no new updates

Effect:
- focus on atmosphere
- deepen world
- build tension slowly

---

## HOW TO USE

Each Telegram post should:

1. Read current UTC block
2. Apply matching tone
3. Blend updates into that tone
4. Continue existing story threads
